# FB Reels Downloader

Beautiful dark-themed web GUI to download Facebook Reels.

## Unraid Installation

1. Install **Docker Compose Manager** plugin
2. Create new stack → paste the `docker-compose.yml`
3. Deploy

Access: `http://YOUR-UNRAID-IP:8000`

## Features
- Modern dashboard UI
- Mobile responsive
- Auto monitoring
- Settings persistence
- etc.
